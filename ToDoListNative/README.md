# Проект

Добро пожаловать в проект! Этот файл содержит инструкции по настройке и началу работы с проектом.

## Перед началом работы

Перед тем как приступить к работе с проектом, вам необходимо указать логин и пароль для вашей локальной базы данных. 

Укажите в файле конфигурации "appsettings.json" все необходимые данные для подключения к локальной базе данных.
